Salvus
=========
```
Tu ne cede malis sed contra audentior ito.
```

Safely `set`, `get` and `remove` any properties in any object. *Anywhere.*


### API

 * **.noto()**
 * **.lego()**
 * **.erado()**
 * **.purgo()**
